# Superstore Sales Analysis

## Project Overview

This project analyzes Superstore sales data to identify key business insights related to sales performance, profitability, products, categories, regions, and customer segments.

## Tools Used

* Excel (Data Cleaning)
* MySQL (Data Analysis)

## Business Questions Answered

1. Which year had the highest sales?
2. Which month had the highest sales?
3. Which category generated the highest profit?
4. Which products generated the highest sales?
5. Which products caused losses?
6. Which region generated the highest sales?
7. Which state and city generated the highest revenue?

## SQL Analysis

Performed analysis using:

* GROUP BY
* SUM()
* HAVING
* ORDER BY
* LIMIT

## Key Insights

* Technology was the most profitable category.
* Several products generated negative profit despite strong sales.
* Sales showed consistent growth over the years.
* The West region generated the highest revenue.
* Profit margin remained positive across most categories.


## Project Outcome

The project demonstrates the complete data analytics workflow from data cleaning and SQL analysis insights generation.
